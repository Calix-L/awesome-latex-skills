# Security Policy

## Reporting a Vulnerability

This project contains AI agent prompt files (markdown). There is no executable code, server, or API.

If you discover a security concern (e.g., a prompt injection vector in a SKILL.md), please email zhenxinlin290@gmail.com.

## Scope

- SKILL.md and reference files: prompt-level issues that could cause harmful agent behavior
- Agent configs (yaml): misconfiguration risks
- Test suite: shell injection risks in test runner
